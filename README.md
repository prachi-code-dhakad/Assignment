
# Assignment


##  Project 1 – Webpage Scraper

Takes a URL from the command line, downloads the webpage, and extracts the title, clean body text, and all links. Uses BeautifulSoup for HTML parsing.

**Tech Used:** Python, requests, BeautifulSoup

**Run:**
pip install requests beautifulsoup4
python scraper.py https://example.com

**Learning:** Web scraping basics, HTTP requests, HTML parsing, command-line input handling.

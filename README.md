# Assignment


##  Project 1 – Webpage Scraper

Takes a URL from the command line, downloads the webpage, and extracts the title, clean body text, and all links. Uses BeautifulSoup for HTML parsing.

**Tech Used:** Python, requests, BeautifulSoup

**Run:**
pip install requests beautifulsoup4
python project1.py https://example.com

**Learning:** Web scraping basics, HTTP requests, HTML parsing, command-line input handling.

##  Project 2 – Document Similarity Checker

**Description:**
Compares two documents by processing their text, hashing words, and generating fingerprints using SimHash to detect similar or near-duplicate content.

**Tech Used:** Python, built-in libraries, hashing techniques

**Run:**
python project2.py <url1> <url2>

**Learning:** Text processing, hashing concepts, and efficient document comparison.
